OGP画像について
================

SNSでシェアされた時に表示される画像です。
以下の仕様で画像を作成し、このフォルダに「ogp.png」として保存してください。

- サイズ: 1200 x 630 ピクセル
- 形式: PNG または JPG
- 内容例: クラブ名「エシャンズ」+ 「秩父市 社会人バドミントンクラブ」+ クラブの写真やイラスト

※ og:image の URL は公開後のドメインに合わせて index.html 内を更新してください。
  例: <meta property="og:image" content="https://yoursite.com/images/ogp.png">
